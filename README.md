# Landing-Page-NutriBroow
Landing page responsiva para o Nutri Broow, nutricionista focado em acompanhamento prático e personalizado. Desenvolvida com React, TypeScript, Vite e Tailwind CSS, possui seções institucionais, benefícios, jornada, FAQ e formulário validado que abre o WhatsApp com mensagem pronta para captação de leads.
